发现使用这个博客做记录还挺不错的，直接在 markdown 里写好，用#做区分，会自动在右侧做出摘要，可以点击跳转阅读，体验也不错，不用每天写一点就新建一篇文章了，毕竟也不是每天都有那么多事要写。

枚举的价值以及如何在 JS 中模拟枚举值—-> 序列与反序列化

枚举，一种从事先定义好的一组常量转换而来的遵从制约的变量

在拥有枚举的类型安全的语言中，枚举被理解为常量，也就是枚举成员以及它们的值都不会发生变化，而在 JS 中，虽然我们可能会重写一个变量为任何的类型，但我们有 Object.freeze() 方法可以将一个对象进行冻结，从而达到和常量一样的效果。

枚举的一个重要的作用是提高可读性

在 JS 这种弱类型的语言中模拟枚举最大的困难在于如何让枚举成员及枚举成员的值像强类型语言的常量一样不变。
