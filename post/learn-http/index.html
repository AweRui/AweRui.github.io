HTTP 历史

很多事情就是这样，面对缩写时会觉得好难理解，一旦把晦涩的缩写展开就会发现里面隐藏的秘密。

HTTP 的中文全称为超文本传输协议，从名字你就能看出来，这就是个传输文本的协议，重点在文本，这就是如今人们说 HTTP 比较慢的原因，相比于传输二进制数据，传输文本当然会慢不少。

但这也怪不得 HTTP，毕竟人家刚被设计出来时就是用来传输文本的，而且也没想到后来会发展得这么快，需要传输的数据量越来越多，图片、音频、视频都用它来传输，能不慢吗？
