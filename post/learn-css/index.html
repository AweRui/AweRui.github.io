CSS 变量

记得之前 CSS 是不支持变量的，这次在查资料的过程中才发现现在 CSS 已经支持变量了

CSS 变量的作用域就是它所在的选择器的有效范围，由于这个原因，全局变量通常放在根元素 :root 里面，确保任何选择器都可以读取它们，如果实在小程序里则是放在 page {} 里面

CSS 变量还可以和响应式布局结合，如1200节点定义一个变量，某属性使用这个变量，到 550 节点时再改变这个变量的值，就可以完成部分响应式

但不是所有浏览器都支持 CSS 变量，对于不支持的浏览器，可以正常写一个属性值，然后在正常属性值的下面使用变量

js 设置变量: document.body.style.setProperty(‘–xxx’, ‘xxx’)

js 读取变量: document.body.style.getPropertyValue(‘–yyy’).trim()

js 删除变量: document.body.style.removeProperty(‘–xxx’)

通过将无效语句写在 CSS 变量里，CSS 变量提供了 js 和 css 通信的一种途径（虽然不知道有什么用）
