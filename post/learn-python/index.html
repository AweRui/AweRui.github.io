Learn Python

2019-11-29

Contents

有位同学最近想学python，问我我当然是支持，但支持归支持，总得有理论支持啊，所以写了点文字来整理自己的想法。

做事之前一定要明确为什么要做这件事，这样才能让自己的行为有支持，才能持久下去，不然就是三秒的热情，还不如不做这件事，那么对于从没接触过编程的人来说，为什么要开始学习编程呢？

我的理由是你需要一件趁手的工具去折叠时间。

折叠时间这个概念是我最近在刷《进击的智人》这本书时读到的，这是本讲人类进化史的书，但别急着否定，这本书并不像你的高中历史课本一样无趣，相反，作者的写作风格相当随和，就像一个朋友在和你聊天一样。

这本书有很多有意思的部分，其中有一章提出的折叠时间这个概念非常有趣。虽然如今的人类和其他动物比起来似乎无所不能，并在很多方面有压倒性的优势，但这种情况出现的时间真的很短，也就是最近一两百年的事，而彻底摆脱饥荒我们至今仍然没有做到，当然除了我们中国（在摆脱饥荒以及扶贫这件事上，中国最近几十年的成绩值得全世界学习）。

想象一下，面对同一块坚硬的骨头或坚果，其他动物想要食用它们需要发育出坚硬的牙齿，巨大的咬合力，这会在资源匮乏的时代大量占用动物身体发育的能量，从而使得它们用来发育大脑的能量大为减少，没有了大脑的充分发育，其他很多事情都会没有可能，如制作精良的工具，发展出精细的语言。而人类则不同，因为我们的祖先学会了使用工具，如一块尖锐的石头，尽管这不能和如今我们精细繁多的工具相比，但仍然有其价值，一块有棱角的石头再粗糙，也能帮原始人更轻松地应对骨头，从而让人类将更多的能量投入到大脑的发育中，而这一点点的差异，最终在未来为人类带来巨大的收益。

另一种情况出现在人类的扩张过程中。当人类走出非洲，想高纬度地区迁徙时，寒冷不可避免地成为了人类必须面对的问题，如果是其他动物想要在高纬度生存，那么不可避免地需要演化出一身厚厚的皮毛来适应寒冷的环境，而这一过程可能需要上百万年的时间。但对于人类来说，将野兽的皮毛剥下来披在自己身上则是个近乎开挂的行为，因为原始人轻易地将其他动物百万年的演化成果折叠起来，披在自己身上，于是人类得以更轻松地向其他地区扩张，并最终在全球留下自己的痕迹。

虽然这些故事发生在久远的原始社会，但仍然对现代人类有启发。今天我们仍然在无意识地继承祖先的行为，不断地制造更多的工具，也在使用更多的工具，我们生产汽车拓展我们的行动半径，我们生产冰箱拓展食物的保存时间，我们利用计算机拓展我们的计算能力，我们在很多领域都在折叠时间，但还不够，因为我们面对的世界已经不再是单纯的物理世界了，我们更多地处在网络世界中，而在网络世界中，我们使用的各种软件虽然也在为我们折叠时间，但我们有时更需要自己制造属于我们自己的趁手工具，自己去主动地“折叠”时间，这就是编程的价值。

但对于从没接触过编程的人来说，他不仅需要一件可以制造工具的工具，这个工具还需要足够简单，也足够强大，思来想去，目前也就只有 python 符合这一点了。

首先，python 的语法足够简洁亲近，基本能当英语读，其次，python 的库足够多，这意味着很多事情都有现成的解决方案，多数情况下你只要引入一个库，然后再自己写几行代码就能有不错的结果。别人花了很长时间完成的一个功能，现在打包成一个库，你只需要轻松引入，就可以拥有别人要花几个月才能拥有的处理能力，这和智人从野兽身上剥下兽皮，披在自己身上，从容应对严寒不是一样令人称奇吗？
